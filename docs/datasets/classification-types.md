## Fracture Classification System

Documents the fracture classification scheme used in this project.

**Content to include:**

- Definition of each fracture type/class being detected
- Clinical descriptions and relevance
- Rationale for classification choice over alternatives
- Visual examples where applicable

**Proposed structure:**

- **Class 0**: Normal (no fracture)
- **Class 1:** Hairline/stress fracture
- **Class 2:** Partial fracture
- **Class 3:** Complete fracture
- **Class 4:** Comminuted/fragmented

**References:** Links to clinical papers or references defining these categories.
